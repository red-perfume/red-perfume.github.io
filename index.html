Red Perfume

"Red Perfume" is a group of open source projects, primarily focused on Atomizing CSS:

  • Red-Perfume-Task-Runner A simplified and focused task runner for websites/webapps.
  • Red-Perfume-CSS A library that takes in any arbitrary CSS and returns an atomized version, including a JSON "classMap" for HTML atomization and dynamic JavaScript.
  • Red-Perfume-HTML A library that takes in any arbitrary HTML and the JSON "classMap" from Red-Perfume-CSS and atomizes the markup.
  • Red-Perfume-Desktop Eventually we plan on creating a desktop app version of Red Perfume as well, written with Vue.js and NW.js.
